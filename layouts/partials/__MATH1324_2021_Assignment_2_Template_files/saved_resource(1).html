Introduction

  • The purpose of this assessment for using Hypthesis Testing to evaluate and compare Baidu machine translation and Google machine translation. The basic idea is using same sentiment analysis tool analyze both side (origin data and translated data) for determine the quality of translation. The high quality of translation should have less differences and high correlation between original data sentiment score and translated data sentiment score.

  • There are two datasets we are used in this research. The primary dataset from kaggle which about Movie audience comments with rating and comments on Personal Tailor (a 2013 Chinese film), collected from Douban (a popular Chinese social media platform).

  • Second dataset, include translated data and Sentiment data (translation Origin Comments to translated Comments, after sentiment analysis both sides).

  • We used these two data sets and merge them together as a new main data set. However, this is an untidy data set, we reshape the data set into a tidy format.

  • We used paired t-test to check the difference of translation tools and Pearson correlation test to compare which tool is better.



Problem Statement

  • In this assignment, we compare Original Comments Score and Translated Comments Score to analyse whether the two translation tool (Google, Baidu) is based on the correctness of the original translation.

  • First, we need to exploring the mean, median, max, min, and IQR for original data Sentiment score and translation data Sentiment score from Google and Baidu translation tools, then draw the boxplots to evaluate the difference about original data Sentiment score and translated data Sentiment score for both Baidu and Google data sets.

  • Then, we will do the paired t-test to check the difference between the original data Sentiment score and translated data Sentiment score by Baidu or Google translation tools.

  • At the last, we would like to check the correlation about original data Sentiment score and translated data Sentiment score by Baidu or Google translation, and compared which translation tools is translated more correlative to original data.

Hypothesis Testing

  • Perform paired t-test to analysis the data set
  • We can assumed that:
    • H0 = the mean of Original Comments Score = the mean of Translated Data Score
    • H1 = the mean of Original Comments Score != the mean of Translated Data Score
  • From the result, we can reject the null hypothesis(H0)
# Compute t-test for google and baidu data set about original comment score and translated data score
GoogleTest <- t.test(Google$OriginCommentsScore, Google$translatedDataScore, paired = TRUE, alternative = "two.sided")
BaiduTest <- t.test(Baidu$OriginCommentsScore, Baidu$translatedDataScore, paired = TRUE, alternative = "two.sided")

Discussion

  • From the t-test, we know that the translated data from Google and Baidu has significant difference from original comments data
  • So, we have checked the relationship between original data and translated data by two translation tools, which are positive relationship.
  • Then, we have tested the correlation by these two variables by using Pearson correlation test, and the p-values of the test for both translation tools are 2.2e-16, which is less than the significance level alpha = 0.05. So, we can conclude that translated data and original data are significantly correlated with a correlation coefficient of 0.50 for Baidu translation tool and 0.527 for Google translation tool.
  • The correlation for google translation is 0.027 higher than Baidu translation, so we can say that Google translation tool is a little bit better than Baidu translation tool by translating Chinese to English.
