## Install
```bash
yay -S clang lldb
```
## Enable Debug Symbols

gcc:
```bash
c++ -g demo.cpp
```

Clang:
```bash
clang++ -g demo.cpp
```

CMAKE:
```bash
cmake -DCMAKE_BUILD_TYPE=Debug
```
